$ mask-agent · 실행 데모

한 명령어로 탐지 → 마스킹 → 검증까지

1

실행

샘플 문서를 mask-agent에 넘긴다.

2

탐지 · 적용

민감 필드를 식별하고 규칙별로 가린다.

3

검증

누락·오탐을 diff로 바로 확인한다.

# 1. 에이전트 실행

$ python -m mask_agent --input samples/voc-memo.md --out out/masked.md

[agent] loading document… ok (2.1 KB · 3 sections)

[agent] detecting sensitive fields…

  · name = "김연습" → rule: person_name

  · rrn = "991332-1234567" → rule: korean_rrn

[agent] applying mask rules… ok (5 fields)

 

# 2. 결과 비교

$ diff samples/voc-memo.md out/masked.md | head

− 주민등록번호: 991332-1234567

+ 주민등록번호: ■■■■■■-■■■■■■■

데모는 한 장면에 한 명령이 원칙 — 길면 나눠 보여주세요.
