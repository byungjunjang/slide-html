에이전트를 직접 돌려보기.

진행 순서

프롬프트 하나로 탐지 → 마스킹 → 저장까지 이어지는 최소 파이프라인.

1

가상환경 활성화 후 mask-agent를 샘플 문서와 함께 실행.

2

에이전트가 문서의 의미 단위를 파악하고 민감 필드를 식별.

3

마스킹 규칙을 적용한 결과를 out/masked.md로 저장.

# 가상환경 활성화

$ source .venv/bin/activate

 

# 에이전트 실행

$ python -m mask_agent \

    --input samples/voc-memo.md \

    --out out/masked.md
